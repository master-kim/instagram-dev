# INSTAGRAM-DEV

- 프로젝트 상태 : 진행중
- 진행중 프로젝트 기간 : 2022년 10월 12일 ~ 

# 프로젝트 참여 개발자 (개발인원:4명)

- 김요한 팀장 (역할분담 : 풀스택)
- 이강현 팀원 (역할분담 : 백엔드)
- 노유미 팀원 (역할분담 : 백엔드)
- 김영일 팀원 (역할분담 : 프론트엔드)

# 핵심 기능
- 로그인, 회원가입 (롬복을 이용한 유효성 검증)
- 팔로우, 팔로잉 (전체 팔로우 추천 보기 등)
- 좋아요, 댓글
- 개인 프로필, 유저 프로필
- 게시글 등록(첨부파일 포함), 수정, 삭제, 조회, 검색
- 스토리 등록(첨부파일 포함), 조회

# 개발 환경
- 개발환경 : Windows 10
- 개발도구 : Eclipse
- 구성환경 : Spring boot, React.js, React-router, JPA, Hibernate, MariaDB
